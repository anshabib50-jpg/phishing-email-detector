# Phishing Email Detector

This project detects phishing emails using:
- Rule-based engine (V2)
- Machine Learning model (V3)

## Features
- Risk scoring
- Confidence percentage
- Real-time prediction

## Technologies
- Python
- Scikit-learn
- Pandas
